# Brainstorm Skill

## 概述
结构化头脑风暴助手，通过多维度引导激发创意，并自动整理想法。

## 功能特性
- 💡 多维度创意激发
- 📝 结构化思维导图
- 🎯 可行性评估
- 📋 行动项提取

## 使用场景
- 产品功能构思
- 解决方案探索
- 内容创作策划
- 问题拆解分析

## 工作流程
```
1. 明确主题/问题
2. 多维度发散（技术/用户/商业/创新）
3. 归类整理想法
4. 评估优先级
5. 输出行动清单
```

## 维度框架
- 技术可行性
- 用户需求
- 商业价值
- 创新程度
- 实施成本
