# Git Push Skill

## 概述
自动化的Git代码推送助手，支持SSH密钥配置、冲突处理和GitHub同步。

## 功能特性
- 🔑 SSH密钥生成与配置
- 🚀 自动commit和push
- 🔄 智能拉取合并冲突
- 📊 推送状态报告

## 使用场景
- Skill开发版本管理
- 配置文件同步

## 工作流程
```
1. 检查Git状态
2. 配置SSH密钥（如需要）
3. 添加GitHub公钥
4. 执行推送
5. 返回推送结果
```

## 依赖工具
- git
- ssh-keygen
- ssh-keyscan
